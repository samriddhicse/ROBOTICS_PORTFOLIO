## Projects

**Autonomous Maze Solver**  
A* pathfinding algorithm on a 2D grid with real-time visualization. My first algorithm project — understanding how robots plan paths.

**ROS2 Custom Package**  
Complete ROS2 package with custom nodes, messages, services, and launch files. The foundation for every robotics project.

**Gazebo Robot Navigation**  
Differential drive robot navigating autonomously using Nav2. Includes teleoperation, odometry, and AMCL localization.

**3D Transformation Visualizer**  
Interactive tool exploring rotation matrices, quaternions, and Euler angles. Understanding how robots think about 3D space.

**Camera Calibration & ArUco Detection**  
Classical computer vision for robot perception. Detecting markers, estimating pose, and understanding camera geometry.

**SLAM Robot**  
Building 2D occupancy grids using LiDAR. Comparing SLAM Toolbox vs Cartographer. The skill most asked in robotics interviews.

**Robot Arm with Inverse Kinematics**  
6-DOF manipulator with interactive IK GUI. Understanding how robot arms move and reach for objects.

**Object Detection & Grasping**  
YOLOv8 detects objects from camera feed. Robot arm picks them up using MoveIt2. Vision + manipulation in one system.

**Human Trust Study**  
Simple user study measuring how robot navigation errors affect human trust. Using NASA-TLX and Schaefer Trust Scale.

**Reinforcement Learning for Grasping**  
Training a robot arm to grasp objects using SAC and HER in MuJoCo. My first steps into AI for robotics.
